Expense Report Extension

This extension allows users to create and manage expense reports directly in Pilot.

Features

  • Add multiple expense rows
  • Automatic amount calculation
  • Tax / VAT per line
  • Currency selection
  • Live total calculation
  • CSV export

How to Use

  1. Open the Expenses menu
  2. Add one or more expense lines
  3. Enter rate, quantity, and tax
  4. Review the total amount
  5. Export the report as CSV if needed

Notes

All calculations are performed locally in the browser. No data is stored on the server unless backend submission is enabled.
